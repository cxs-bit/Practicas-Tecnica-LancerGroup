# Comandos importantes

### Docker
Iniciar Container
php artisan app:run-dc

Detener 
php artisan app:stop-dc

# Setup del projecto
composer install
npm i
npm run dev

# Server
php artisan key:generate
php artisan migrate:fresh --seed
php artisan serve
